# Source paper

This protocol operationalizes one published study. Cite it when reporting any
result produced by this skill.

> Tang, C.; Sittplangkoon, C.; Xiang, C.; Schnur, L.; Duan, R.; Lin, X.; Li, D.;
> Yao, Z. **Evaluating Osteoarthritis Severity in Mice Using µCT-Derived
> Geometric Indices.** *Biology* **2026**, *15*, 262.
> https://doi.org/10.3390/biology15030262

- Corresponding author: Zhenqiang Yao, University of Rochester Medical Center.
- Published 31 January 2026, MDPI. Open access, **CC BY 4.0** — figures reused as
  reference assets under that license, attributed here.
- Vendored copy: `assets/source-paper.pdf`. Supplemental Tables S1–S3 (per-model
  ROC cutoffs, per-specimen values, tibia rotation corrections) are in the
  paper's supplement, not vendored here.

**What this skill takes from the paper:** the two geometric indices and their
operational landmark definitions (§2.3), the scanner and threshold parameters
(§2.2), the structure set and the sesamoid→osteophyte confounder (Introduction,
Figure 2A), and the phenotype cutoffs as *interpretation context* (Figures 2–4,
Table 1) — never as gates on a running measurement.
